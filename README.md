# 龙高北重开模拟器

> 一个模拟初中三年生活的文字剧情游戏

## 游戏简介

《龙高北重开模拟器》是一款模拟经营类文字剧情游戏，玩家将重新经历初中三年的校园生活，体验学业成长、活动参与、人际关系等多元化玩法，最终以中考和毕业典礼为终点。

## 核心玩法

- **学业系统**：语文、数学、英语、政治、历史、物理、化学、生物、地理、体育等多科目学习与考试
- **时间线系统**：完整模拟初中三年六个学期的校园生活
- **活动系统**：社团活动、运动会、艺术节等校园活动参与
- **人际关系**：同学社交、舍友互动、师生关系
- **心理系统**：青春期心理事件、暗恋、压力管理等
- **关键事件**：分班、月考取消、台风预警、签约龙高等

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式与响应式布局
- **JavaScript (ES6+)**：游戏逻辑
- **LocalStorage**：数据持久化

## 开发方式

采用 **Vibe Coding** 模式开发，强调快速迭代、随时修改、专注内容而非架构。

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。
